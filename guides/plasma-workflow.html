Illustrator DXF exports are broken. Adobe segments arcs into hundreds of tiny line fragments. Always use SVG from Illustrator — SheetCam reads it at 1:1 scale with clean geometry.

DXF vs SVG — When to Use Each

✦ DXF

  • Use from: SolidWorks, AutoCAD, QCAD
  • Must be R12/R14 with polylines + arcs
  • True arcs = cleanest geometry possible
  • Industry standard for engineering drawings
  • ⚠ Splines cause jagged cuts in SheetCam

✦ SVG

  • Use from: Illustrator, Inkscape, web graphics
  • Imports at 1:1 scale — no sizing issues
  • Fewer nodes = smoother toolpaths
  • Best for artistic/sign work & logos
  • SheetCam handles SVG very well
Rule of thumb: Engineering parts → DXF from SolidWorks. Art, signs, logos → SVG from Illustrator or Inkscape.

SolidWorks → DXF Quick Steps

Export Procedure
  • Right-click flat face → Export to DXF/DWG
  • Set format to R12 (broadest compatibility)
  • Check "Splines as Polylines"
  • Check "High Quality DWG" + "Merge Points"
  • Verify scale is 1:1 with correct units (in or mm)
  • Remove all dimensions, notes, and title blocks
Illustrator → SVG Procedure
  • Select all → Type → Create Outlines
  • Select all → Object → Path → Simplify
  • File → Export As → SVG (not "Save As")
  • Profile: SVG 1.1 · Fonts: Convert to Outlines

Pre-Flight Checklist

  • All text converted to outlines / paths
  • All geometry = closed paths (no open endpoints)
  • No stray points, construction lines, or dimensions
  • File is correct scale (1:1) and units match
  • Inside cuts on separate layer from outside profiles
  • Verify in SheetCam: measure a known dimension
Pro Tip: Design on any workstation → save to USB as DXF or SVG → do SheetCam only at the plasma station. This mirrors how real fabrication shops operate.
